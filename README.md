# Netfix-Analysis

# The data set downloaded from data.world consists of TV shows and movies on Netflix as of 2021. It consists of below attributes:
 
1. Show ID
2. Type
3. Title
4. Director
5. Caast
6. Country
7. Date added on Netflix
8. Release year
9. Rating
10. Duration
11. Genre
12. Description
